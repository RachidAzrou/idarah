Implementation notes

SVG math: function polarToCartesian(cx, cy, r, angleDeg) + describeArc(cx,cy,r,startAngle,endAngle) to draw a path with A r r 0 largeArcFlag 1 x y.

Convert percent to degrees with sweep = clamp(percent/100 * 300, 0, 300). Start angle = -60°, end angle = start + sweep.

Radii: outer 110, middle 86, inner 62 (adjust to 280×280 viewBox).

Stroke-linecap = round; stroke-width = 14 (outer), 14 (middle), 14 (inner) for visual consistency.

Add a light white mask at the center to create a “ring” look (or separate background strokes).

Numbers locale: format counts with Intl.NumberFormat('nl-BE').

Colors meet WCAG AA against white background.

Mobile: when width < 768px, place total above legend, chart left; keep adequate spacing.
